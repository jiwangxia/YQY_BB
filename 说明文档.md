# 1.文件作用：

## 1.1   .vcxproj.filters作用

```
YQY (项目)
├── 📁 Header Files（头文件）
│   └── YQY.h
├── 📁 Source Files（源文件）  
│   ├── main.cpp
│   └── YQY.cpp
├── 📁 Form Files（界面文件）
│   └── YQY.ui
└── 📁 Resource Files（资源文件）
    └── YQY.qrc
```

只能用于显示，便于分类，实际上还是同一个文件夹

```
YQY\YQY\
├── main.cpp
├── YQY.cpp
├── YQY.h
├── YQY.ui
└── YQY.qrc
    ↑ 全部在同一个文件夹里！
```
